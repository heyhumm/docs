Linear is the issue tracking tool that streamlines software projects, sprints, tasks, and bug tracking.

## Tools

| Tool | Description |
|------|-------------|
| `LINEAR_LIST_ISSUES` | List issues with filtering and pagination |
| `LINEAR_GET_ISSUE` | Get details for a specific issue |
| `LINEAR_LIST_PROJECTS` | List accessible projects |
