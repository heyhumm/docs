Mixpanel is a product analytics platform that helps teams analyze how users engage with their products.

## Tools

| Tool | Description |
|------|-------------|
| `MIXPANEL_QUERY_EVENTS` | Query event data with filters and aggregations |
| `MIXPANEL_GET_FUNNELS` | Get funnel conversion data |
| `MIXPANEL_GET_RETENTION` | Get user retention cohort data |

## Authentication

<Card title="API Secret" icon="key">
  Service account authentication with API secret. Create a service account in Mixpanel Project Settings.
</Card>
