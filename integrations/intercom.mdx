Intercom is a customer service solution that uses AI, bots, and human support to help businesses deliver personalized customer experiences.

## Tools

| Tool | Description |
|------|-------------|
| `INTERCOM_SEARCH_CONVERSATIONS` | Search conversations with filters |
| `INTERCOM_LIST_CONTACTS` | List contacts with pagination |
| `INTERCOM_GET_CONTACT` | Get details for a specific contact |
