HubSpot is a customer platform with software, integrations, and resources to connect marketing, sales, and customer service.

## Tools

| Tool | Description |
|------|-------------|
| `HUBSPOT_LIST_OBJECTS` | List objects with pagination and property filtering |
| `HUBSPOT_READ_RECORD` | Read a specific record by ID |
| `HUBSPOT_SEARCH_OBJECTS` | Search records with filters, sorting, and pagination |
