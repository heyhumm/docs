ChurnZero is a real-time customer success platform that helps subscription businesses fight customer churn.

## Tools

| Tool | Description |
|------|-------------|
| `CHURNZERO_LIST_ACCOUNTS` | List accounts with filtering and pagination |
| `CHURNZERO_LIST_CONTACTS` | List contacts with filtering and pagination |
| `CHURNZERO_LIST_EVENTS` | List customer events and activities |
| `CHURNZERO_GET_CHURN_SCORES` | Get churn risk scores for accounts |
