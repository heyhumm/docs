Pendo helps product teams understand and guide users to create product experiences that customers love.

## Tools

| Tool | Description |
|------|-------------|
| `PENDO_RUN_AGGREGATION` | Run aggregation queries on product usage data |
| `PENDO_GET_DISTINCT_METADATA_VALUES` | Get unique values for metadata fields |

## Authentication

<Card title="API Key" icon="key">
  Integration key authentication. Obtain your key from Pendo Settings > Integrations > Integration Keys.
</Card>
