PostHog is the open-source product analytics platform that helps engineers build better products.

## Tools

| Tool | Description |
|------|-------------|
| `POSTHOG_CREATE_QUERY` | Execute HogQL queries against your PostHog project |

## Authentication

<Card title="API Key" icon="key">
  Personal API key authentication. Generate a key from PostHog Project Settings > Personal API Keys.
