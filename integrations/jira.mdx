Jira is a project management tool that helps teams plan, track, and manage software development.

## Tools

| Tool | Description |
|------|-------------|
| `JIRA_SEARCH_ISSUES` | Search issues using JQL queries |
| `JIRA_GET_ISSUE` | Get details for a specific issue |
| `JIRA_LIST_PROJECTS` | List accessible projects |
